# 작업 로그

Claude와의 작업 내역을 기록하는 디렉토리입니다.

## 파일 명명 규칙

```
YYYY-MM-DD-작업명.md
```

예시:

- `2025-01-01-패치데이터검증.md`
- `2025-01-02-크롤링로직개선.md`

## 작업 로그 템플릿

```markdown
# [작업명]

## 개요

- **시작일**: YYYY-MM-DD
- **상태**: 진행중 | 완료 | 중단
- **관련 브랜치**: feature/XX

## 목표

- [ ] 목표 1
- [ ] 목표 2

## 진행 상황

### YYYY-MM-DD

- 완료한 작업 내용
- 발견한 이슈

## 실행 스크립트

### 1단계: [설명]

\`\`\`bash
npx tsx scripts/example.ts
\`\`\`

### 2단계: [설명]

\`\`\`bash
npx tsx scripts/example2.ts --option
\`\`\`

## 중단 시 이어서 할 작업

- [ ] 다음에 해야 할 작업 1
- [ ] 다음에 해야 할 작업 2

## 관련 파일

- `scripts/example.ts`
- `data/example.json`
```
